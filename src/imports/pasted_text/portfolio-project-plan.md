# Build an Award-Winning Premium AI Portfolio Website

You are a senior team consisting of a UI/UX Designer, Frontend Architect, Backend Engineer, Motion Designer, and Product Designer.

Your task is to create a world-class, premium, futuristic portfolio website that feels like the portfolio of a top AI Engineer and Software Developer.

The website should look as if it was designed by Apple, Linear, Stripe, Framer, Vercel, and Awwwards-winning designers.

The design must feel cinematic, elegant, luxurious, modern, minimal, interactive, and highly polished.

---

## Primary Goal

Build a fully responsive portfolio website with an integrated Admin Dashboard where I can edit everything without touching the code.

Every piece of content must be editable from the admin panel.
